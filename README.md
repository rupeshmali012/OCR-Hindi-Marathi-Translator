# OCR-Hindi-Marathi-Translator
A Python GUI application for English OCR and Hindi/Marathi translation using PaddleOCR and Deep-translator.

# Image Text Translator & OCR System

Developed a professional GUI-based application to extract English text from images and translate it into regional languages. 


Key Features

- High-Accuracy OCR: Integrated PaddleOCR for robust English text extraction from images. 



- Multilingual Translation: Translates extracted text into Hindi and Marathi using Deep Translator. 



- User Interface: Built with Tkinter, featuring image previews and a custom application icon. 



- Real-time Status: Includes a status bar to track processing stages (e.g., "Processing... Please wait"). 



- Utility Tools: Options to save extracted text as .txt files or copy to clipboard for ease of use. 


Tech Stack

- Language: Python 



- OCR Engine: PaddleOCR 



GUI: Tkinter 



Libraries: PIL (Pillow), Deep-translator, Pyperclip
